Add your real 3D component models here:

- `gpu.glb`
- `cpu.glb`
- `ram.glb`

`PcPreview3D` will auto-load these files when present.
If a file is missing or cannot be loaded, it falls back to the built-in procedural model.

Recommended export settings:

- Format: `glTF Binary (.glb)`
- Up axis: `Y`
- Apply transforms/freeze scale before export
- Keep mesh centered close to origin for easier placement
